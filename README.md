# Songloft LX Music Plugin (lxmusic)

为自托管音乐服务器 [Songloft](https://songloft.ai) 提供洛雪音乐 (lx-music) 生态的第三方音源集成。

## 免责声明

1. 本插件**不附带任何音源脚本**。用户需自行导入第三方洛雪音源 `.js` 文件。
2. 音源脚本与其数据均为第三方版权内容，仅供个人学习研究使用。
3. 使用本插件搜索、获取的歌曲链接及歌词等内容，用户须自行承担版权责任。
4. 禁止用于商业用途。如有侵权，请及时删除相关数据。

## 构建

```bash
npm install
npm run build
npm run validate
```

产物位于 `dist/lxmusic.jsplugin.zip`。
